# 🎲 Random String Generator

A React.js application that generates random strings dynamically using React Hooks such as **useState**, **useCallback**, and **useEffect**. The application provides a simple and interactive interface for creating unique random strings with optimized performance and efficient state management.

## 🚀 Features

- Generate random strings instantly
- Built entirely with React Functional Components
- State management using `useState`
- Function optimization using `useCallback`
- Lifecycle management using `useEffect`
- Responsive and user-friendly interface
- Real-time string generation
- Lightweight and fast performance

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Vite
- CSS / Tailwind CSS

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ridd-coder/random-string-generator.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
